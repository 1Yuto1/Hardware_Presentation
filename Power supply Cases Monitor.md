## Introduction

Good morning/afternoon, everyone.

Today, we‚Äôll be exploring three essential components that make up the foundation of every computer system ‚Äî the **power supply**, the **computer case**, and the **monitor**.

Each of these parts plays a vital role in how a computer operates and interacts with the user.

- The **power supply** is the heart of the system, delivering stable and reliable electricity to every component.

## Power Supply

## üï∞Ô∏è History
- **Early 1900s:** Power conversion was done with basic transformers and rectifiers.  
- **1950s‚Äì1960s:** Linear regulated power supplies offered more stable voltage.  
- **1970s:** Switch-Mode Power Supplies (SMPS) appeared‚Äîsmaller, lighter, and more efficient.  
- **2000s‚ÄìToday:** Efficiency standards like **80 PLUS**, modular cabling, and quiet fans became standard.

## ‚öôÔ∏è Main Functions
1. **Convert Power:** Changes AC from the wall to DC for computer components.  
2. **Regulate Voltage:** Keeps voltage stable despite input or load changes.  
3. **Protect Components:** Safety features like over-voltage and short-circuit protection.  
4. **Distribute Power:** Supplies correct voltage to the motherboard, CPU, GPU, and drives.

## Key Specifications

| Specification | Description |
|----------------|--------------|
| **Wattage (W)** | Total power output capacity (e.g., 550W, 750W, 1000W) |
| **Efficiency Rating** | Power conversion efficiency (80 PLUS Bronze, Gold, Platinum) |
| **Form Factor** | ATX, SFX, or TFX (physical size) |
| **Rails** | +12V, +5V, +3.3V voltage lines |
| **Connectors** | ATX 24-pin, CPU 8-pin, PCIe, SATA, Molex |
| **Cooling** | Usually includes a fan for heat dissipation |

# Computer Case 

## üï∞Ô∏è History
- **1970s‚Äì1980s:** Simple metal desktop enclosures.  
- **1990s:** Tower cases offered better airflow and expandability.  
- **2000s:** Tool-less designs, windows, and better cooling emerged.  
- **2010s‚ÄìPresent:** Tempered glass, RGB lighting, and modular designs are common.

## Main Functions
1. **Protection:** Shields components from dust and damage.  
2. **Cooling:** Manages airflow and mounts for fans/radiators.  
3. **Organization:** Holds components like the motherboard, PSU, and GPU.  
4. **Accessibility:** Allows easy upgrades and cable management.  
5. **Aesthetics:** Adds a personal style to the system.

## Key Specifications

| Specification | Description |
|----------------|--------------|
| **Form Factor** | Supports ATX, Micro-ATX, Mini-ITX motherboards |
| **Drive Bays** | 2.5‚Äù/3.5‚Äù bays for SSDs and HDDs |
| **Cooling Support** | Fan/radiator mounts and airflow design |
| **GPU Clearance** | Max graphics card size supported |
| **Front I/O Ports** | USB, audio, power/reset buttons |
| **Materials** | Steel, aluminum, tempered glass |

# Monitor

## History
- **1950s‚Äì1990s:** CRT monitors were standard‚Äîheavy but vivid.  
- **2000s:** LCD screens replaced CRTs‚Äîlighter, energy-efficient.  
- **2010s:** LED backlighting and IPS panels improved brightness and color.  
- **2020s‚ÄìPresent:** OLED, Mini-LED, curved, and ultrawide displays dominate.

## Main Functions
1. **Visual Output:** Displays text, images, and video.  
2. **User Interaction:** Acts as the computer‚Äôs visual interface.  
3. **Color & Clarity:** Provides accurate and detailed visuals for work, gaming, and media.

## Key Specifications

| Specification | Description |
|----------------|--------------|
| **Screen Size** | Diagonal measurement in inches |
| **Resolution** | Pixel count (1080p, 1440p, 4K) |
| **Panel Type** | TN (fast), IPS (accurate), VA (contrast), OLED (self-lit) |
| **Refresh Rate** | Updates per second (60Hz‚Äì360Hz) |
| **Response Time** | Speed of pixel change (lower = better) |
| **Brightness** | Measured in nits |
| **Ports** | HDMI, DisplayPort, USB-C, VGA |
| **Color Gamut** | sRGB, DCI-P3, AdobeRGB color range |
